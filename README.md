# TableEasy

TableEasy é um sistema para facilitar a criação, visualização e gerenciamento de tabelas de forma prática e intuitiva. Ideal para usuários que precisam organizar dados de maneira rápida e eficiente.

## Funcionalidades

- Criar novas tabelas
- Editar e deletar tabelas existentes
- Pesquisar e filtrar dados
- Exportar tabelas para CSV/Excel
- Interface amigável e responsiva
